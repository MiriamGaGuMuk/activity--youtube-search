# Youtube Search

## Setup

1. In Terminal:
  ```sh
  #(1) create a directory for this project in `~/muktek/warmups` and cd into it.
  mkdir ~/muktek/warmups/activity--youtube-search
  cd ~/muktek/warmups/activity--youtube-search
  ```

2. Create your project files and folders
  ```
  - index.html
  |

    |
    - style.css
  ```
